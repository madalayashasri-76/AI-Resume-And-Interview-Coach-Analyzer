# 🤖 AI Interview Coach

## 📌 Project Overview

AI Interview Coach is an AI-powered web application designed to help students and job seekers prepare for technical interviews.

The application analyzes resumes, extracts important skills, calculates a resume score, and generates personalized interview questions, ideal answers, and interview tips using **Mistral AI**.

This project combines **Artificial Intelligence, Natural Language Processing, and Streamlit** to create an interactive interview preparation platform that helps users improve their interview skills and career readiness.

---

# 🚀 Features

## 📄 Resume Analyzer

- Upload resume in PDF format
- Extract resume text automatically
- Detect technical skills
- Analyze resume sections
- Calculate resume score
- Provide resume improvement suggestions

---

## 🤖 AI Interview Preparation

- Select target job role
- Generate personalized interview questions
- Generate ideal answers
- Provide interview preparation tips
- Get role-based interview guidance using Mistral AI

---

## 📊 Resume Evaluation

- Skill identification
- Resume completeness check
- Resume section analysis
- Resume score calculation
- Improvement recommendations

---

## 📥 Report Generation

- Generate interview preparation report
- Download AI-generated interview report
